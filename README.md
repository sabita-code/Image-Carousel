# Image-Carousel
This is a simple **Image Carousel** built purely with **HTML,CSS and Javascript** 

---
### Features
<ul>
  <li>Automatic Sliding: Images transition automatically every few seconds.</li>
  <li>Active Dot Indicators: Navigation dots below the carousel highlight the current active slide.</li>
  <li>Manual Navigation: Previous/Next arrows for user control.</li>
</ul>

---
### Tech Stack
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
